# movie
电影网站    

首页  后台录入    电影详情    电影列表（查看、更新、删除）  用户列表   
路由  
    
/   首页      
/movie/:id      电影详情页   
/admin/movie/new    后台电影录入页       
/admin/movie/list     后台电影列表页   
/admin/user/list    后台用户列表页
技术支持          
nodejs/express  pug  bootstrap  mongodb/mongoose  grunt   bower   npm  express-session
