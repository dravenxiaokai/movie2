# Movie 电影网站    

## 功能分析
1. 首页
2. 后台录入
3. 电影详情
4. 电影列表（查看、更新、删除）
5. 用户列表   
6. 路由  
    
### 路由导航
1. /   首页      
2. /movie/:id      电影详情页   
3. /admin/movie/new    后台电影录入页       
4. /admin/movie/list     后台电影列表页   
5. /admin/user/list    后台用户列表页

### 技术栈          
1. nodejs/express
2. pug
3. bootstrap
4. mongodb/mongoose
5. grunt
6. bower
7. npm scripts
8. express-session
